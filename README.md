# claims-agent

A portfolio project: a HIPAA-shaped, member-scoped agentic AI assistant that drafts plain-English explanations of synthetic health insurance claims. **All members, claims, and policy documents in this repository are synthetic.** No real PHI exists anywhere in this codebase. The synthetic-data banner in the UI is not decorative; it is load-bearing.

Built on AWS Bedrock (Claude Haiku 4.5), LangChain.js v1, Pinecone, DynamoDB, and React.
